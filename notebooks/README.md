# Notebook Roadmap

Recommended notebook sequence:

## `01_data_audit.ipynb`

- load data
- inspect columns and types
- measure missing values
- inspect target distribution
- flag likely leakage variables

## `02_eda_prfeatures.ipynb`

- descriptive statistics
- univariate plots
- merged vs. not merged comparisons
- correlations and pairwise relationships

## `03_supervised_baselines.ipynb`

- feature selection
- train/test strategy
- baseline models
- metric comparison
- error analysis

## `04_unsupervised_analysis.ipynb`

- standardized feature subset
- clustering experiments
- cluster interpretation
- visualizations

## `05_final_story.ipynb`

- cleaned narrative
- final figures
- conclusions
- limitations
- next steps

Each notebook should end with a short interpretation section in plain language, not just code and charts.
